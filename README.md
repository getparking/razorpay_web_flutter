# rzpwplugin

A RazorPay Web JS wrapper
Can be used for POC. 

To use this plugin kindly add your SSH key to Github user settings
follow this link :
https://help.github.com/en/enterprise/2.15/user/articles/adding-a-new-ssh-key-to-your-github-account
